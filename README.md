# EduConnect Frontend

Modern React frontend interface for EduConnect, a multi-college student portal management system.

## Technologies
- React
- Vite
- Tailwind CSS
- React Router DOM
- React Icons
- Framer Motion

## Setup
1. Open the `educonnect-frontend` folder in your code editor.
2. Install dependencies with `npm install`.
3. Start the dev server with `npm run dev`.

## Notes
- This project is frontend-only and uses mock data.
- Supabase integration is prepared in `src/services/supabase.js` but not implemented.
